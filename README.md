# P5X Gacha Calculator

A probability calculator and simulator for P5X. Allows you to simulate and forecast different scenarios to estimate whether your resources are enough to cover pull plans.

## Features

- **Banner Types** - Support for both the Chance (80 Pity with 50/50) and the Targeted (110 Pity but guaranteed) character banners.
- **Success Rate Analysis** - See your odds of fully completing your planned pull schedules without any failures
- **Failure Point Tracking** - Identify which banners most commonly fail, allowing for better planning and adjustment.
